# Eppo SDK Relay App

This app connects to an Eppo Packing Test Runner Server and relays assignment requests to the SDK.

## Running

Build and run the app

```shell
./gradlew installDebug && \
  adb shell am start -n cloud.eppo.android.sdkrelay/.MainActivity 
```
